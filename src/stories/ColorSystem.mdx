# Hans UI Color System

This page documents `src/styles/colors.scss` and shows visual swatches for palette tokens and theme combinations.

## Structure

- `900`: strong
- `500`: default
- `300`: neutral
- `gray` also has `700` and `100`
- `base` semantic mapping uses gray scale:
- `--base-strong-color` = `gray-700`
- `--base-default-color` = `gray-500`
- `--base-neutral-color` = `gray-300`
