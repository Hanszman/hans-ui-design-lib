# Hans UI Color System

This page documents `src/styles/colors.scss` and shows visual swatches for all palettes and all theme combinations.

## Structure

- `900`: strong
- `500`: default
- `300`: neutral
- `gray` also has `700` and `100`

Semantic tokens used by components:
- `--primary-*`
- `--secondary-*`
- `--success-*`
- `--danger-*`
- `--warning-*`
- `--info-*`
- `--base-*`

## Usage Note

Use semantic tokens in component SCSS (`--primary-default-color`, `--base-neutral-color`, etc.) instead of raw family tokens (`--purple-500`, etc.).
