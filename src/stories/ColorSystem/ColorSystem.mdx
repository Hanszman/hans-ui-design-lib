# Hans UI Color System

## Structure

- `900`: strong
- `500`: default
- `300`: neutral
- `gray` also has `700` and `100`

<ColorSystemDocs />

## Usage Note

Use semantic tokens in component SCSS (`--primary-default-color`, `--base-neutral-color`, etc.) instead of raw family tokens (`--purple-500`, etc.).

## Theme Selection

The current approach remains supported:

```html
<html data-theme="combination3"></html>
```

The five built-in combinations are still the default way to pick a preset palette for a project.

## Dynamic Combination Object

Projects that need a custom palette can override the semantic tokens at runtime with a complete combination object containing the same 23 color tokens used by the built-in combinations.

### React / npm

```tsx
import { setHansTheme } from 'hans-ui-design-lib';

setHansTheme({
  primary: {
    strong: '#1e3a8a',
    default: '#2563eb',
    neutral: '#bfdbfe',
  },
  secondary: {
    strong: '#9f1239',
    default: '#e11d48',
    neutral: '#fecdd3',
  },
  success: {
    strong: '#166534',
    default: '#22c55e',
    neutral: '#bbf7d0',
  },
  danger: {
    strong: '#991b1b',
    default: '#ef4444',
    neutral: '#fecaca',
  },
  warning: {
    strong: '#92400e',
    default: '#f59e0b',
    neutral: '#fde68a',
  },
  info: {
    strong: '#155e75',
    default: '#06b6d4',
    neutral: '#cffafe',
  },
  base: {
    strong: '#334155',
    default: '#64748b',
    neutral: '#cbd5e1',
  },
  backgroundColor: '#f8fafc',
  textColor: '#0f172a',
});
```

    primary: {

      default: '#2563eb',
      neutral: '#bfdbfe',

      default: '#e11d48',
      neutral: '#fecdd3',

    success: {
      strong: '#166534',
      default: '#22c55e',
      neutral: '#bbf7d0',
    },
    danger: {

      default: '#ef4444',

    },
    warning: {
      strong: '#92400e',
      default: '#f59e0b',
      neutral: '#fde68a',

      strong: '#155e75',
