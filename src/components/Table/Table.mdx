The **Table** component renders reusable data grids with optional sorting and per-column filtering.

## Usage

```tsx
import { HansTable } from 'hans-ui-design-lib';

const columns = [
  { key: 'name', header: 'Name', sortable: true },
  { key: 'role', header: 'Role' },
];

const rows = [
  { name: 'Ana', role: 'Editor' },
  { name: 'Bruno', role: 'Viewer' },
];

export const Example = () => <HansTable columns={columns} rows={rows} />;
```

## Sortable

## Filterable

Set `filter` per column using either `input` or `dropdown` mode.  
For dropdown filters, a clear option is added automatically. You can customize it with `clearLabel` (column) or `defaultDropdownFilterClearLabel` (table).
