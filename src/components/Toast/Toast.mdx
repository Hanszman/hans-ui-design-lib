The **Toast** component renders fixed notification cards without external libraries. It supports semantic colors, visual variants, stack-aware placement in any corner, optional dismissal and automatic timeout handling.

## Usage

```tsx
import { HansToast } from 'hans-ui-design-lib';

export const Example = () => (
  <HansToast
    title="Profile updated"
    message="The new data is already available."
    toastColor="success"
    toastVariant="neutral"
    position="top-right"
  />
);
```

## Positions

Use `position` to move the toast to any corner. The stack automatically grows downward for top positions and upward for bottom positions.

<Canvas of={HansToastStories.Positions} />

## Stacked Notifications

Multiple visible toasts in the same corner are measured and offset vertically to avoid overlap.
