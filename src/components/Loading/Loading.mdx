The **Loading** component supports two modes:

- `spinner`: circular, compact loading indicator
- `skeleton`: block placeholder that can fill the parent area

## Usage

```tsx
import { HansLoading } from 'hans-ui-design-lib';

export const Example = () => (
  <HansLoading loadingType="spinner" loadingColor="primary" />
);
```

Then, you can use the component in your template. Here is a basic example:
