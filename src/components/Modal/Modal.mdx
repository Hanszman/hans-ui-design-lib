The **Modal** component renders dialog layers through a portal so it can escape local layout containers in apps and Storybook. It supports centered dialogs, left and right side panels, optional header/body/footer sections, semantic color variants and custom children with vertical or horizontal scrolling when needed.

## Usage

```tsx
import { useState } from 'react';
import { HansButton, HansModal } from 'hans-ui-design-lib';

export const Example = () => {
  const [open, setOpen] = useState(false);

  return (
    <>
      <HansButton label="Open modal" onClick={() => setOpen(true)} />
      <HansModal
        isOpen={open}
        onOpenChange={setOpen}
        title="Create workspace"
        confirmLabel="Create"
        cancelLabel="Cancel"
        modalColor="primary"
        modalVariant="outline"
      >
        <div>Any custom content can be rendered here.</div>
      </HansModal>
    </>
  );
};
```
