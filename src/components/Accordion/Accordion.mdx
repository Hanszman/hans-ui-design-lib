The **Accordion** component displays a list of text sections that can be expanded
and collapsed one by one. It is designed for FAQ-style content and uses
`IoIosArrowUp` and `IoIosArrowDown` to communicate the current state of each item.

## Usage

```tsx
import { HansAccordion } from 'hans-ui-design-lib';

const items = [
  {
    id: 'shipping',
    title: 'How long does shipping take?',
    description: 'Shipping usually takes between 3 and 5 business days.',
  },
];

export const Example = () => (
  <HansAccordion items={items} defaultOpenItemIds={['shipping']} />
);
```
