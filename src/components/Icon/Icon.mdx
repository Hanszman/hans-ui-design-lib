The **Icon** component dynamically renders any icon from the [React Icons](https://react-icons.github.io/react-icons/) library using only the icon name.

It allows consuming projects (React, Angular, etc.) to use icons **without directly importing** `react-icons`, acting as a middleware for consistent usage across frameworks.
