The **Card** component is a reusable visual template for structured content blocks. It supports a profile-style layout with avatar, title and description, and an image-cover layout where the artwork fills the entire card area with text layered above it.

## Usage

```tsx
import { HansCard } from 'hans-ui-design-lib';

export const Example = () => (
  <HansCard
    title="Design sync"
    description="Avatar + text composition for kanban items and dashboards."
    avatarSrc="https://i.pravatar.cc/120?img=24"
  />
);
```
