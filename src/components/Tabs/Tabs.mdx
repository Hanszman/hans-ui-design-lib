The **Tabs** component organizes content by sections and lets users switch between them.

## Usage

```tsx
import { HansTabs } from 'hans-ui-design-lib';

const tabs = [
  { id: 'overview', title: 'Overview', content: <p>Overview content</p> },
  { id: 'details', title: 'Details', content: <p>Details content</p> },
];

export const Example = () => <HansTabs tabs={tabs} />;
```

Then, you can use the component in your template. Here is a basic example:

## Colors

Use `tabsColor` to style the active tab.
When `tabsColor="base"`, it falls back to the primary style.

<Canvas of={HansTabsStories.Colors} />

## Closable Tabs

Enable `showCloseButton` to show the close icon (`IoIosCloseCircle`) on each tab.

<Canvas of={HansTabsStories.ClosableTabs} />

## Per Tab Close

Use `closable: true` in specific tabs to make close optional per tab.
