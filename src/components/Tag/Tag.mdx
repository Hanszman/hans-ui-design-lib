The **Tag** component shows small labels with optional removable action.

## Usage

```tsx
import { HansTag } from 'hans-ui-design-lib';

export const Example = () => (
  <HansTag label="Tag" actionIcon="MdClose" onAction={() => {}} />
);
```

Then, you can use the component in your template. Here is a basic example:
