The **Dropdown** component can work as a simple select, a multi-select, and an
autocomplete input depending on its configuration.

## Usage

```tsx
import { HansDropdown } from 'hans-ui-design-lib';

const options = [
  { label: 'Option 1', value: 'opt-1' },
  { label: 'Option 2', value: 'opt-2' },
];

export const Example = () => (
  <HansDropdown
    label="Dropdown"
    options={options}
    selectionType="single"
    enableAutocomplete
  />
);
```
