The **Dropdown** component is a button-triggered popup menu.  
It supports option lists by default and custom content through `children`.
By default, the trigger uses `triggerShape="square"` and can be changed to rounded when needed.

## Usage

```tsx
import { HansDropdown } from 'hans-ui-design-lib';
```
