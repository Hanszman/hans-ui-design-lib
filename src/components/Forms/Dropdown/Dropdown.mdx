The **Dropdown** component can work as a simple select, a multi-select, and an
autocomplete input depending on its configuration.
Options can include an `id` for selection values and an optional `imageSrc` to
display an avatar in the list.

## Usage

```tsx
import { HansDropdown } from 'hans-ui-design-lib';

const options = [
  { id: 'opt-1', label: 'Option 1', value: 'opt-1' },
  { id: 'opt-2', label: 'Option 2', value: 'opt-2' },
];

export const Example = () => (
  <HansDropdown
    label="Dropdown"
    options={options}
    selectionType="single"
    enableAutocomplete
  />
);
```
