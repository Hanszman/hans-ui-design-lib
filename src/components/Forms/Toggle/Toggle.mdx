The **Toggle** component switches a single setting between on/off states.

## Usage

```tsx
import { HansToggle } from 'hans-ui-design-lib';

export const Example = () => (
  <HansToggle
    label="Enable notifications"
    checked
    toggleColor="primary"
  />
);
```
