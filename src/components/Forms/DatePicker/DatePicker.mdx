The **DatePicker** component supports `date`, `datetime` and `time` selection while reusing the Hans input, icons and popup patterns.

## Usage

```tsx
import { HansDatePicker } from 'hans-ui-design-lib';

export const Example = () => (
  <HansDatePicker
    label="Start date"
    pickerType="datetime"
    inputColor="primary"
    calendarColor="secondary"
    defaultValue="2026-03-13T14:30"
  />
);
```
