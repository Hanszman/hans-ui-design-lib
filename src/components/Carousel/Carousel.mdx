The **Carousel** component displays image galleries with previous and next
navigation, progress indicators, loading skeletons and an empty state.
Each slide can optionally render a title and description above the image, using
the same visual direction as the image layout from the card component.

## Usage

```tsx
import { HansCarousel } from 'hans-ui-design-lib';

const items = [
  {
    id: 'cover-1',
    title: 'Campaign',
    description: 'A highlighted hero image.',
    imageSrc: 'https://example.com/cover.jpg',
    imageAlt: 'Campaign cover',
  },
];

export const Example = () => (
  <HansCarousel
    items={items}
    visibleItemsCount={1}
    carouselColor="primary"
    carouselVariant="outline"
  />
);
```
