The **Avatar** component displays a circular profile image.
If image loading fails (or no `src` is provided), it falls back to a user icon.

## Usage

```tsx
import { HansAvatar } from 'hans-ui-design-lib';

export const Example = () => (
  <HansAvatar src="https://my-cdn.com/user.png" alt="User avatar" />
);
```
