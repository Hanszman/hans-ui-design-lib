The **Chart** component renders charts through Apache ECharts using a typed API.

Reference documentation from the original library:
**https://echarts.apache.org/en/index.html**

You can use that link to explore more chart examples and then map them to `optionOverrides` when needed.

## Usage

```tsx
import { HansChart } from 'hans-ui-design-lib';

export const Example = () => (
  <HansChart
    chartType="mixed"
    categories={['Jan', 'Feb', 'Mar']}
    series={[
      { name: 'Orders', type: 'bar', data: [12, 18, 16], label: { position: 'inside' } },
      { name: 'Revenue', type: 'line', data: [20, 24, 22], label: { position: 'diagonal' } },
    ]}
    colors={['primary', 'rgb(255, 140, 66)']}
  />
);
```

## Types

Supported chart modes include `line`, `bar`, `pie`, `doughnut`, and `mixed`.

In `mixed`, use per-series `type` to combine bars and lines in one chart.

<Canvas of={HansChartStories.MixedBarLine} />

## Labels

Labels are configured **per series** with `series[].label`.

Supported positions:
- `horizontal`
- `vertical`
- `diagonal`
- `inside`
- `none`

Examples with and without labels:

## Colors

Color behavior:
- Pass DS tokens (`primary`, `secondary`, `success`, etc.) to use active theme colors.
- Pass custom colors (`rgb(...)`, hex, etc.) for custom palettes.
- Pass `colors={[]}` to randomize from design-system combination colors.
