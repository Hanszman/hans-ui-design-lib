The **Pagination** component renders previous, numbered and next navigation controls using the same semantic button tokens from the design system. It is reusable on its own and also powers modal footers that need fixed page navigation without recreating local pagination shells in each consumer app.

## Usage

```tsx
import { HansPagination } from 'hans-ui-design-lib';

export const Example = () => (
  <HansPagination
    currentPage={2}
    totalPages={5}
    firstLabel="First"
    previousLabel="Previous"
    nextLabel="Next"
    lastLabel="Last"
    pageLabel="Page"
    onPageChange={(page) => alert(page)}
  />
);
```
